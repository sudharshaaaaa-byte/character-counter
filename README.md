# Character Counter

A simple and responsive Character Counter built using **HTML, CSS, and JavaScript**. This project updates the number of typed and remaining characters in real time as the user types into the text area.

## Features
- Real-time character counting
- Displays typed characters
- Displays remaining characters
- Maximum character limit of 200
- Clean and responsive user interface
- Beginner-friendly JavaScript project

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Structure
```
Character-Counter/
│── index.html
└── style.css
```
